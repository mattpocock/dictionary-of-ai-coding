The relevant information the [agent](./Agent.md) has access to right now. The abstract noun — not the buffer (that's the [context window](./Context%20window.md)), not the running history (that's the [session](./Session.md)), but *what the agent knows that's pertinent to the task*. "Loading something into context" means making it part of this set; "context engineering" is the discipline of curating it.

*Usage:* "It keeps inventing fields that aren't in the type." "The type file isn't in context — it's reading the call sites and guessing. Read the definition in first."
